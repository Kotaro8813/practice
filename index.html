HTML & CSS

HTMLでWebページの土台ができます。CSSでデザインやレイアウトが変更でき、普段、私達が見ているWebページを作ることができます。

jQuery

JavaScriptと呼ばれるプログラミングを扱いやすくしたライブラリです。Webページに動きを付けるアニメーションなど色々なことができます。

Ruby

読みやすく書きやすいシンプルな言語です。Webアプリケーションなど様々なソフトウェアを作ることができます。

PHP

スクリプト言語と呼ばれ、HTMLに埋め込むことができます。見る人によって異なるページが表示される動的なページを作ることができます。
